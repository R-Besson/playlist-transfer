    How to get your Spotify Client ID:
    
					
      
						
    1. Go to the Spotify

    How to get your YouTube API Key:
    
					
      
						
    1. Go to the Google Cloud
								Console and create a project.
      2. 
						
    2. Go to APIs & Services > Library. Search for and
							Enable the YouTube Data API v3.
						
      3. 
						
    3. Go to APIs & Services > Credentials. Click +
								CREATE CREDENTIALS and choose API Key.
      4. 
					
    
				
    
			
    
			
    
				
    
					
    How to get your YouTube API Key & Client ID:
    
					
      
						
    1. Go to the Google Cloud
								Console and create a project.
      2. 
						
    2. Go to APIs & Services > Library. Search for and
							Enable the YouTube Data API v3.
						
      3. 
						
    3. Go to APIs & Services > OAuth consent screen >
							Audience and add your email as a Test User.
						
      4. 
						
    4. Go to APIs & Services > Credentials and click +
								CREATE CREDENTIALS.
      5. 
						
    5. Create an API Key.
      6. 
						
    6. Create an OAuth client ID for a Web application and add
							this page's address (e.g., https://r-besson.github.io/playlist-transfer/) to
							both Authorized JavaScript origins and Authorized redirect
								URIs.
      7.

      4. 
						
    4. Refresh the page. Click on any request in the list and go to Headers >
							Request Headers.
						
      5.

      6. 
						
    6. Copy the entire value for the authorization header (it starts with "Bearer

      7.
